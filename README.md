# Home Automation

Système domotique complet avec ESP32, contrôle d'éclairage RGB et monitoring de capteurs.

## Contenu

Voir le README complet dans le dossier `Domotic/`

## Démarrage rapide

```bash
cd Domotic
cp .env.example .env
nano .env  # Configurer DOMAIN, passwords, email
./deploy.sh production
```
